# 💻 Methu Portfolio Website

## 📌 Project Title  
**Personal Portfolio Website – Methul (AI & Web Developer Portfolio)**

---

## 🛠️ Technologies Used  

- HTML5 – Structure of the website  
- CSS3 – Custom styling and animations  
- Bootstrap 5.3 – Responsive layout and grid system  
- JavaScript (Vanilla JS) – Interactivity and dynamic behavior  
- Font Awesome – Icons  
- Google Fonts (Poppins) – Typography  

---

## 🚀 Features Implemented  

### 🎨 UI & Layout  
- Modern dark-themed UI design  
- Gradient color styling for headings and buttons  
- Clean and professional layout  
- Smooth scrolling navigation  

### 📱 Responsiveness  
- Fully responsive design using Bootstrap grid system  
- Works on Desktop, Tablet, and Mobile devices  

### 🧭 Navigation  
- Sticky navbar with scroll spy  
- Active section highlighting while scrolling  
- Smooth section transitions  

### 🏠 Hero Section  
- Introduction with name and role  
- Call-to-action buttons (View Work, Hire Me)  
- Social media links  
- Animated profile/avatar  

### 👤 About Section  
- Personal introduction  
- Skills displayed using badges  
- Statistics cards (projects, experience, etc.)  

### ⚙️ Skills Section  
- Skill cards with icons  
- Progress bars with percentages  
- Animation using Intersection Observer  

### 📂 Projects Section  
- Project cards with title and description  
- Technology tags  
- Links for Live Demo and Source Code  
- Hover effects and animations  

### 📬 Contact Section  
- Contact information (email, phone, location)  
- Contact form UI  
- Social media links  

### ✨ Extra Features  
- Scroll-based active navigation highlight  
- Hover animations and transitions  
- Floating animation for profile image  
- Clean reusable CSS classes
